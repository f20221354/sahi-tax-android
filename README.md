Sahi Tax: India's First AI Tax OptimizerSahi Tax is an automated Tax-Loss Harvesting (TLH) platform designed specifically for retail investors in India. By leveraging AI and a deterministic rule engine, it identifies and executes opportunities to save on capital gains tax, a service previously reserved for High Net Worth Individuals (HNIs).  🚀 The ProblemAwareness Gap: 99% of retail investors are unaware that Tax-Loss Harvesting exists.  Complexity: Tracking 8-year loss carry-forwards (under Section 74 of the IT Act) across multiple brokers is nearly impossible for manual tracking.  Timing Failure: Traditional CAs file in July, long after the March 31st deadline for tax-saving trades has passed.  Cost Gap: Existing PMS firms charge ₹2-5L/year, making them unaffordable for the 1.5 crore retail investors filing ITR-2.  ✨ Key FeaturesAutomated Tax-Loss Harvesting: Identifies losing positions to offset booked capital gains.  Deterministic Rule Engine: Encodes Section 74/94 nuances (carry-forwards and stripping rules) with CA-audited logic.  AI Explanation Layer: Uses the Claude API to provide human-friendly explanations for every tax-saving recommendation.  One-Tap Execution: Direct integration with major brokers like Zerodha (Kite Connect) for seamless trades.  ITR Ready Reports: Automatically generates Schedule CG for ITR-2 filings.  🛠️ Tech StackFrontend: Next.js, Tailwind CSS, shadcn/ui (Mobile-first PWA).  Backend & Rule Engine: Deterministic Python engine for tax logic.  AI Integration: Claude API for decision explanation.  Data Parsing: Python + pdfplumber for Consolidated Account Statement (CAS) processing.  Data Infrastructure: Sahamati Framework (via Finvu) for RBI-approved Account Aggregator consent.  📈 User FlowConnect: Upload CAS PDF or connect via Account Aggregator/Broker APIs.  Select: View flagged tax-saving opportunities and select stocks to liquidate.  Generate: Review a dynamic "Tax Optimization Report".  Execute & File: One-tap trade execution and download ITR-ready schedules.  🛡️ Trust & ComplianceSEBI Compliance: Designed for SEBI-RIA (Investment Advisor) partnership to ensure legal advisory standards.  Security: Encrypted data handling through RBI-regulated Account Aggregator rails.  Auditability: Core logic is audited by professional CA firms to ensure compliance with the latest Income Tax Act sections.  🗺️ RoadmapPhase 1: CAS PDF Upload for instant coverage.  Phase 2: Account Aggregator (AA) integration for automated tracking.  Phase 3: Direct Broker APIs (Zerodha, Upstox, ICICI) for live execution.  Expansion: Moving beyond TLH to Salary Optimization and continuous AI-driven CFO services.  Developed by Mirnal Gupta (BITS Pilani).
